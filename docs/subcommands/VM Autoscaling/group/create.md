# VmAutoscalingGroupCreate

## Usage

```text
ionosctl vm-autoscaling group create [flags]
```

## Aliases

For `vm-autoscaling` command:

```text
[vmas vm-as vmasc vm-asc vmautoscaling]
```

For `group` command:

```text
[g groups]
```

For `create` command:

```text
[c]
```

## Description

Create a VM-Autoscaling group
