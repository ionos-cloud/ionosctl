# VmAutoscalingGroupDelete

## Usage

```text
ionosctl vm-autoscaling group delete [flags]
```

## Aliases

For `vm-autoscaling` command:

```text
[vmas vm-as vmasc vm-asc vmautoscaling]
```

For `group` command:

```text
[g groups]
```

For `delete` command:

```text
[d del rm]
```

## Description

Delete VM Autoscaling Groups
