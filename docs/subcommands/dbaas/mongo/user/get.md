# DbaasMongoUserGet

## Usage

```text
ionosctl dbaas mongo user get [flags]
```

## Aliases

For `mongo` command:

```text
[mongodb mdb m]
```

For `get` command:

```text
[g]
```

## Description

Get a MongoDB user

## Options

```text
  -i, --cluster-id string   The unique ID of the cluster
  -d, --database string     The authentication database
  -u, --user string         The authentication username
```

## Examples

```text
ionosctl dbaas mongo user get
```
