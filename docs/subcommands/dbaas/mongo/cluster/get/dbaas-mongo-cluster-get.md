# DbaasMongoClusterGet

## Usage

```text
ionosctl dbaas mongo cluster get [flags]
```

## Aliases

For `mongo` command:

```text
[mongodb mdb m]
```

For `cluster` command:

```text
[c]
```

For `get` command:

```text
[g]
```

## Description

Get a Mongo Cluster by ID

## Options

```text
  -i, --cluster-id string   The unique ID of the cluster (required)
```

## Examples

```text
ionosctl dbaas mongo cluster get --cluster-id <cluster-id>
```
