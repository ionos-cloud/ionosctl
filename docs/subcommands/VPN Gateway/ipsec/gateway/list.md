# VpnIpsecGatewayList

## Usage

```text
ionosctl vpn ipsec gateway list [flags]
```

## Aliases

For `gateway` command:

```text
[g gw]
```

For `list` command:

```text
[l ls]
```

## Description

List IPSec Gateways
