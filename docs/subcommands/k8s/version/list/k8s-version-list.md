# K8sVersionList

## Usage

```text
ionosctl k8s version list
```

## Aliases

For `version` command:

```text
[v]
```

For `list` command:

```text
[l ls]
```

## Description

Use this command to retrieve all available Kubernetes versions.

## Examples

```text
ionosctl k8s version list
```
