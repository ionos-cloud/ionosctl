# K8sVersionGet

## Usage

```text
ionosctl k8s version get
```

## Aliases

For `version` command:

```text
[v]
```

For `get` command:

```text
[g]
```

## Description

Use this command to retrieve the current default Kubernetes version for Clusters and NodePools.

## Examples

```text
ionosctl k8s version get
```
